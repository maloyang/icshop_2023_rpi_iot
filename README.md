# icshop_2023_rpi_iot

## 物聯網MQTT應用 - 以樹莓派為例

課程簡介：本課程將以MQTT實作物聯網之應用，將從樹莓派基礎操作進行介紹
  ，進而使用Python程式語言進行MQTT的應用，及物聯網資訊的收集應用
  ，最後使用NodeRED快速完成物聯網的視覺化應用
